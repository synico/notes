
### date
#### unix timestamp转时间
```
date -d '@1460537339'
```
#### 时间转unix timestamp
```
date +%s -d '2020-04-21 10:54:00'
```
#### 当前时间转unix timestamp
```
date +%s
```

### 查看进程占用内存
#### top
首先通过ps命令找到进程id，再通过`top -p pid`查看CPU和内存占用率。
#### ps
查看CPU和内存占用率，以及物理内存使用量 (单位是k) 。
```
ps aux | grep java
```
####  文件/proc/pid/status
通过查看进程的status文件，根据项VmRSS查看物理内存使用情况。
```
cat /proc/pid/status
```
